piano-play
==========

##基于html5的钢琴弹奏游戏

- demo地址：http://www.xuanfengge.com/piano


##功能

- 看谱弹琴
- 录制曲子
- 分享曲子
- 弹奏试听

##2016-12-10更新

- 优化代码，抽出为组件
- piano-play.js 可直接被引用或者AMD、CMD方式加载
- 增加动画
- 取消本地保存曲子功能
